# Roadmap

This roadmap tracks prioritized improvements for the OSSAfrica Community repository.

## Near Term (0-3 Months)

- Consolidate and simplify core documentation
- Standardize issue and pull request intake templates
- Add CI checks for markdown and links
- Seed repository with example contributions per category
- Improve contribution discoverability with index sections

## Mid Term (3-6 Months)

- Introduce monthly community highlights from submissions
- Establish contributor spotlight process
- Improve topic tagging and navigation across content
- Expand contributor onboarding resources

## Long Term (6+ Months)

- Regional chapter activity tracking
- Programmatic content catalog generation
- Mentorship pathways for new open source contributors

## How To Propose Roadmap Changes

1. Open an issue with context and expected impact.
2. Discuss tradeoffs with maintainers/community.
3. Submit a pull request updating this file after consensus.
