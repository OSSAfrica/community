# Community Resources

This directory contains shared resources, tools, and materials for the OSSAfrica community.

## What's Included

This directory can contain:

- Learning resources and guides
- Tool recommendations
- Useful links and references
- Community best practices
- Documentation templates
- Development guides
- Career resources

## Adding Resources

To add a resource:

1. Create a markdown file with a descriptive name
2. Include clear descriptions and links
3. Organize by topic or category
4. Submit a pull request

## Categories

Resources can be organized by:

- **Learning** - Educational materials and courses
- **Tools** - Development tools and software recommendations
- **Career** - Job resources and career development
- **Best Practices** - Community standards and guidelines
- **Documentation** - How-to guides and documentation

See the [CONTRIBUTING.md](../CONTRIBUTING.md) for detailed guidelines.
