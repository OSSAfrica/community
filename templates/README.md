# Contribution Templates

This directory contains templates for submitting different types of contributions to the OSSAfrica community.

## Available Templates

- **project-template.md** - For submitting open source projects
- **article-template.md** - For sharing articles and blog posts
- **talk-template.md** - For documenting talks and presentations
- **tutorial-template.md** - For creating tutorials and learning materials
- **event-template.md** - For organizing or promoting events
- **member-template.md** - For adding your community member profile

## How to Use Templates

1. Copy the appropriate template to the correct directory
2. Rename the file with a descriptive name
3. Fill in all required fields
4. Replace placeholder text with your information
5. Remove any optional sections you don't need
6. Submit a pull request

## Template Guidelines

- Fill in all required fields marked with `[REQUIRED]`
- Optional fields can be left out if not applicable
- Use clear and concise language
- Provide accurate information
- Include all relevant links

See the [CONTRIBUTING.md](../CONTRIBUTING.md) for detailed contribution guidelines.
