# Community Members

This directory contains profiles of OSSAfrica community members.

## Adding Your Profile

To add your member profile:

1. Copy the template from `templates/member-template.md`
2. Fill in your information
3. Save the file with your name (e.g., `john-doe.md`)
4. Submit a pull request

## What to Include

- Full name
- Location (city, country)
- Bio (brief introduction)
- Skills and interests
- GitHub profile
- Social media links (optional)
- Website or portfolio (optional)
- Areas of expertise
- What you're looking to learn or collaborate on

## Privacy

Only include information you're comfortable sharing publicly. All fields except name are optional.
