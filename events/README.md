# Community Events

This directory contains information about OSSAfrica community events, meetups, and conferences.

## Adding an Event

To add an event:

1. Copy the template from `templates/event-template.md`
2. Fill in the event information
3. Save the file with the event name and date (e.g., `2024-01-15-community-meetup.md`)
4. Submit a pull request

## What to Include

- Event name
- Date and time (include timezone)
- Location (physical address or virtual meeting link)
- Event description
- Agenda or schedule
- Registration link (if applicable)
- Organizer information
- Event type (meetup, conference, workshop, hackathon, etc.)
- Target audience

## Event Categories

- **Meetups** - Local community gatherings
- **Workshops** - Hands-on learning sessions
- **Conferences** - Larger events with multiple speakers
- **Hackathons** - Coding competitions and collaborative projects
- **Webinars** - Online presentations and discussions
