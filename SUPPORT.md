# Support

## Where To Get Help

For help with contributions, templates, or repository usage:

1. Open a GitHub issue in this repository for public questions.
2. Start a GitHub Discussion when available for broader community conversation.
3. Reach out to maintainers listed in [MAINTAINERS.md](MAINTAINERS.md) for escalations.

## What To Include In Support Requests

- A clear summary of your question or problem
- Relevant links (PR, issue, contribution file)
- Steps already attempted
- Expected vs actual outcome

## Response Expectations

Maintainers are volunteers. Response times may vary.

- Community questions: usually within 5 business days
- Governance or moderation concerns: priority triage
- Security concerns: follow [SECURITY.md](SECURITY.md)

## Code of Conduct

All support interactions are governed by [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
