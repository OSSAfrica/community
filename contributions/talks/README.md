# Talks and Presentations

This directory contains information about talks, presentations, and speaking engagements by OSSAfrica community members.

## Submitting Your Talk

To add your talk:

1. Copy the template from `templates/talk-template.md`
2. Fill in your talk information
3. Save the file with a descriptive name (e.g., `intro-to-cloud-computing.md`)
4. Submit a pull request

## What to Include

- Talk title
- Speaker information
- Event name and date
- Recording link (if available)
- Slides link (if available)
- Abstract or description
- Topics covered

See the [CONTRIBUTING.md](../../CONTRIBUTING.md) for detailed guidelines.
