# Open Source Projects

This directory contains information about open source projects contributed by OSSAfrica community members.

## Submitting Your Project

To add your project:

1. Copy the template from `templates/project-template.md`
2. Fill in your project information
3. Save the file with a descriptive name (e.g., `my-awesome-project.md`)
4. Submit a pull request

## What to Include

- Project name and description
- Repository URL
- Technology stack
- License information
- How to get started
- Contribution guidelines for your project
- Contact information

See the [CONTRIBUTING.md](../../CONTRIBUTING.md) for detailed guidelines.
