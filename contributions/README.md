# Community Contributions

This directory contains all community contributions organized by type.

## Directory Structure

- **projects/** - Open source projects contributed by community members
- **articles/** - Blog posts, articles, and written content
- **talks/** - Conference talks, presentations, and speaking engagements
- **tutorials/** - Step-by-step tutorials and learning materials

## How to Contribute

Please refer to the main [CONTRIBUTING.md](../CONTRIBUTING.md) file for detailed instructions on how to submit your contributions.

Each subdirectory contains specific contributions of that type. Use the appropriate template from the `templates/` directory when submitting your contribution.
