# Tutorials and Learning Materials

This directory contains tutorials and learning materials created by OSSAfrica community members.

## Submitting Your Tutorial

To add your tutorial:

1. Copy the template from `templates/tutorial-template.md`
2. Fill in your tutorial information
3. Save the file with a descriptive name (e.g., `docker-basics-tutorial.md`)
4. Submit a pull request

## What to Include

- Tutorial title
- Author information
- Difficulty level (Beginner, Intermediate, Advanced)
- Technologies covered
- Prerequisites
- Learning objectives
- Tutorial content or link to external tutorial
- Estimated time to complete

See the [CONTRIBUTING.md](../../CONTRIBUTING.md) for detailed guidelines.
