### Step 1: Clone the repository

```bash
git clone https://github.com/OSSAfrica/community.git
cd community
```

### Step 2: Create a branch

```bash
git checkout -b docs/improve-example
```

### Step 3: Commit and push

```bash
git add .
git commit -m "docs: improve example"
git push origin docs/improve-example
```
