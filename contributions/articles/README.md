# Articles and Blog Posts

This directory contains links and information about articles and blog posts written by OSSAfrica community members.

## Submitting Your Article

To add your article:

1. Copy the template from `templates/article-template.md`
2. Fill in your article information
3. Save the file with a descriptive name (e.g., `getting-started-with-python.md`)
4. Submit a pull request

## What to Include

- Article title
- Author information
- Publication date
- Link to the published article
- Brief summary or abstract
- Tags/topics covered

See the [CONTRIBUTING.md](../../CONTRIBUTING.md) for detailed guidelines.
